Zip Code

  • A project made with the purpose of calculating zip code using the "Via Cep" API to calculate zip codes for different locations
  • I used PHP as the API driver for the main page, and also used it for two validations: checking if the zip code is valid and ignoring special characteres typed
  • HTML for page assembly and also for blocking all entries except for the "postal code" entry field
  • Pure CSS for styling
